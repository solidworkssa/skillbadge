# SkillBadge

NFT-based skill certification and verification system.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/SkillBadge.sol`
- Stacks: `contracts/stacks/contracts/skillbadge.clar`

## Setup
```bash
pnpm install
pnpm dev
```
